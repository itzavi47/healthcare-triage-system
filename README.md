AI Symptom Analyzer & Smart Health Assistant

A lightweight medical triage assistant with OTC guidance, home remedies, and smart reminders.

Overview

This project is a mini healthcare assistant that analyzes user-provided symptoms and generates:

✔️ Possible diseases

✔️ Risk level (Low / Medium / High)

✔️ When to see a doctor

✔️ Emergency warning signs

✔️ Home remedies

✔️ Safe OTC medicine suggestions

✔️ Auto-generated reminders (medicine & doctor visits)

It is built as a Kaggle Notebook / Colab Notebook project, using rule-based NLP + structured medical knowledge.

Features
1. Symptom Detection

Takes symptoms such as fever, body pain, cold, cough, and identifies:

Possible medical conditions

Risk severity

Whether hospitalization is needed

2. Medical Guidance

For each case, the system provides:

Doctor guidance

Emergency red-flag symptoms

Safe OTC medication (Paracetamol, ORS, etc.)

Avoids dangerous or prescription-only drugs

3. Home Remedies

Includes scientifically supported remedies for:

Fever

Body pain

Hydration

Muscle relaxation

Diet suggestions

4. Smart Reminders (Optional Module)

Supports in-notebook notifications:

Desktop notifications

Scheduled medicine reminders

Doctor visit reminders
